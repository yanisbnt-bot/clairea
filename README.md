# Clairea - Plateforme de Gestion Médicale Tout-en-Un

Clairea est une solution open-source moderne et complète conçue pour simplifier la gestion quotidienne des professionnels de santé. L'objectif est de centraliser tous les outils nécessaires au sein d'une interface unique, fluide et sécurisée.

## 🚀 Fonctionnalités Principales
* *Gestion des Rendez-vous :* Prise de rendez-vous en ligne et rappels automatiques.
* *Dossier Patient Informatisé :* Suivi complet des antécédents médicaux et des prescriptions.
* *Coordination des Soins :* Outils de communication sécurisés entre praticiens.

## 🛠️ Stack Technique
* *Frontend :* React.js / Next.js
* *Backend :* Node.js / Express
* *Base de données :* PostgreSQL
